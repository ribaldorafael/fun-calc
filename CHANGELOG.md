# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-05-15

### Added
- Theme selection menu (character-select style)
- 8 calculator themes: Retro Arcade, Destruction, Living Creature, Rube Goldberg, Gravity, RPG Battle, Odontology, Dragon
- Calculator engine with basic and scientific modes
- Expression parser (recursive-descent, no eval)
- Calculation history
- Keyboard support
- Easter eggs (42, big numbers, negative results, zero float-away)
- Sound effects (Web Audio API)
- Canvas particle system (confetti, rain, bursts)
- Rust backend with Tauri 2 (math engine, scientific functions, constants)
- 194 unit tests (120 JS + 74 Rust)
